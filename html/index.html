How it works

Enter any message you would like into the 'Content' section of the form, enter the key that you would like the message to be identified by, optionally check the box if you want your message to be publicly available for 30 days, and click the 'Submit' button.

By default, messages are private, and are only retrievable once. In an effort to preserve the privacy of users' messages, if someone submits a message with the same key as a previously stored message, it is silently overwritten. If you would like your message to be associated with a given key for more than one-time access, you can check the box to make the message public.

No information about users is retained on the server; once you post your message, the only record of it is the message itself.
